# Storybook

Storybook 运行于你的应用程序之外.

因此,你可以孤立地开发UI组件,而不必担心特定于应用程序的依赖性和需求. 运行 npm run storybook 运行storybook开发服务,服务监听于 6006 端口.

构建一个静态storybook副本, 运行 npm run build-storybook.
