# 蜂蜜饮

现在,你希望将应用程序上线,应该快速&最优化.你想要完整的包,PWA包.

在开发模式中,你有非优化代码,源映射以及编写优秀代码所需的所有好东西.

但是在生产中你不希望有不必要的代码,它应该尽可能小.

# 优化构建

要优化你的应用程序以进行生产,请运行命令 npm run build.

这将如下优化你的应用程序:

  • 它运行webpack 生产模式
  • 它为你的应用程序添加了一个服务工作线程提供离线代理服务 来制作PWA
  • 它压缩所有资源
